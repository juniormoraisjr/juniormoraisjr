### 🚀 Sobre mim

- 💻 Back-end com **Java + Spring Boot 3** — APIs REST, microsserviços e integrações
- 📨 Mensageria assíncrona com **Apache Kafka** — produção, consumo e monitoramento
- ☁️ Cloud: **AWS** (Lambda, S3, EKS, DocumentDB, CloudWatch, Glacier)
- 🐳 Containers & Orquestração: **Docker** e **Kubernetes** (probes, HPA, Spring Actuator)
- 🧪 Testes: JUnit 5, Mockito, Testcontainers
- 📐 Arquitetura: Hexagonal (Ports & Adapters), Clean Architecture, REST versionado
- 📈 Interesse em performance, observabilidade e otimização de infraestrutura

---

### 🎯 Atualmente focando em

- 🔧 Otimização de infraestrutura AWS — Lambda, S3 e DocumentDB
- ☸️ Kubernetes: resiliência de pods, probes e HPA em Spring Boot
- 📦 Starters reutilizáveis com Spring Boot 3 Auto-configuration
- 📊 Observabilidade com CloudWatch e métricas estruturadas
